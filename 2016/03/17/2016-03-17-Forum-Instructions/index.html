项目启动步骤

  • 安装IEDA。下载网址，如何破解自己搞定，Good luck！

    http://www.jetbrains.com/idea/download/#section=windows

  • 安装Navicat Premium，安装包及破解包会发给你。

  • 导入数据库，下面介绍导入数据库的方法：
    • 打开之后界面如下：
    • 点击左上角【连接】按钮，新建链接，记住密码！，下面的项目会用到这个！其中端口号、用户名为默认即可。这里我们创建一个名字为newDemo连接。图片如下所示：
    • 新建完成后，右键单击newDemo，选择【新建数据库】，这里新建一个数据库名为medical(名字随便起，你新建的数据库的时候名字叫做jfinal2.0就行)的数据库。记住这个数据库的名称，下面会用到这个数据库名称！图片如下所示：
    • 新建成功后，右键单击medical，【运行SQL文件】，选择根目录 / doc / jfinal2.0.sql，点击【开始】。图片如下：
    • 当看到信息日志里面出现successfully的时候导入成功。图片如下所示：
    • 刷新medical数据库便看到新导入的数据库表了。图片如下所示：

  • 打开IEDA，导入项目：

    • 选择项目

    • 选择maven形式

    • 导入依赖包，下载的会巨慢无比，耐心等待

    • 然后一路next 到最后。
    • IEDA会自动下载项目的maven依赖
    • 导入成功

    • 配置数据库：根目录 > src > main > resources > congig.properties

          jdbcUrl =jdbc:mysql://localhost:3306/jfinal2.0
    user =root //不变
    password =123456
    devMode = true //不变
    showSql = true //不变
    # 项目的访问路径
    base.url = http://localhost:8080

** 修改1：jdbcUrl =jdbc:mysql://localhost:3306/数据库名称（新建数据库时候的数据库名称）**

** 修改2：password =数据库的密码（新建数据库时候的数据库密码）**
