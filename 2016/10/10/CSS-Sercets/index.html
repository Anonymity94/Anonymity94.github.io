outline方案

某些情况下，你可能只需要两层边框，那就可以先设置一层常规边框，再加上outline(描边)属性来产生外层的边框。这种方法的一大优点在于边框样式十分灵活，不像上面的box-shadow方案只能模拟实线边框（假设我们需要产生虚线边框效果，box-shadow就没辙了）

  • 结果预览

  • HTML代码

    <div class="multiple-borders-outline"></div>

<style type="text/css">
    .multiple-borders-outline{
        width: 100px;
        height: 60px;
        margin: 0 auto;
        /* 注意：outline属性实现的边框暂不支持圆角 */
        border-radius: 10px;

        background: yellowgreen;
        outline: 5px solid deeppink;
    }
</style>
  • 知识点
  1. box-shadow

calc()方案

  • 结果预览
calc()方案

  • HTML代码

    <div class="background-position-calc">calc()方案方案</div>

<style type="text/css">
    .background-position-calc{
        background: url('../../images/css-secrets/code-pirate.svg')
                    no-repeat bottom right #58a;
        /* calc()函数 */
        background-position: calc(100% - 20px) calc(100% - 10px);

        height: 200px;
        padding: 10px;
        margin: 0 auto;
        color: white;
    }
</style>
